# Playwright .NET BDD SauceDemo Framework

This project is a Playwright automation framework built with C#, .NET, NUnit, and Page Object Model design pattern. It automates end-to-end workflows for the SauceDemo e-commerce application.

## Tech Stack

- C#
- .NET 10
- Playwright
- NUnit
- Page Object Model
- GitHub
- Git

## Automated Test Scenarios

- Valid user login
- User login and logout
- Add product to cart
- Complete checkout flow

## Framework Structure

- Config - Framework configuration
- Drivers - Playwright browser driver setup
- Pages - Page Object classes
- TestData - Test data files
- Utils - Reusable helper classes
- Reports - Test reports
- Screenshots - Failure screenshots
